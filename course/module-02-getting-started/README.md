# Module 2: Getting Started

Welcome to Module 2! This module will guide you through setting up your development environment and creating your first TanStack Start project.

## Module Overview

In this module, you'll learn:

- What prerequisites you need
- How to install and set up TanStack Start
- How to create your first project
- How the development server works

## Lessons

1. [Prerequisites & Requirements](./lesson-01-prerequisites.md)
2. [Installation & Setup](./lesson-02-installation-setup.md)
3. [Creating Your First Project](./lesson-03-first-project.md)
4. [Understanding the Development Server](./lesson-04-dev-server.md)

## Learning Objectives

By the end of this module, you will:

- Have a complete development environment set up
- Understand the system requirements
- Be able to create new TanStack Start projects
- Understand how the development server works
- Know how to run and debug your application

## Time to Complete

Estimated time: 60-90 minutes

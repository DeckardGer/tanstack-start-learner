# Module 5: Advanced Routing

Welcome to Module 5! This module covers advanced routing concepts in TanStack Start, including type-safe routing, data loading, search parameters, and authentication.

## Module Overview

In this module, you'll learn:

- Type-safe routing with full TypeScript support
- Route loaders and data loading patterns
- Search parameters with validation
- Nested routes and advanced layouts
- Route context and dependency injection
- Route guards and authentication

## Lessons

1. [Type-Safe Routing](./lesson-01-type-safe-routing.md)
2. [Route Loaders and Data Loading](./lesson-02-route-loaders.md)
3. [Search Parameters with Type Safety](./lesson-03-search-parameters.md)
4. [Nested Routes and Advanced Layouts](./lesson-04-nested-routes.md)
5. [Route Context and Dependency Injection](./lesson-05-route-context.md)
6. [Route Guards and Authentication](./lesson-06-route-guards.md)

## Learning Objectives

By the end of this module, you will:

- Leverage full type safety in routing
- Implement efficient data loading strategies
- Work with validated search parameters
- Build complex nested route structures
- Use context for dependency injection
- Implement authentication and authorization
- Create route guards and redirects

## Time to Complete

Estimated time: 150-180 minutes

## Prerequisites

Before starting this module, you should have:

- Completed Module 4: Routing Fundamentals
- Understanding of TypeScript
- Familiarity with async/await

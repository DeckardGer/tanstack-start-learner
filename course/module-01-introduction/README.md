# Module 1: Introduction & Overview

Welcome to Module 1! This module introduces you to TanStack Start and the broader TanStack ecosystem.

## Module Overview

In this module, you'll learn:

- What TanStack Start is and how it works
- The TanStack ecosystem and how the pieces fit together
- Why TanStack Start is a compelling choice for modern web development
- How TanStack Start compares to other popular frameworks

## Learning Objectives

By the end of this module, you will:

- Understand what TanStack Start is and its core purpose
- Know the key libraries in the TanStack ecosystem
- Understand the benefits and trade-offs of using TanStack Start
- Be able to make informed decisions about when to use TanStack Start

# Module 4: Routing Fundamentals

Welcome to Module 4! This module will teach you everything about TanStack Router's powerful file-based routing system.

## Module Overview

In this module, you'll learn:

- How file-based routing works in TanStack Start
- Creating and organizing route components
- Navigation between pages with links
- Working with route parameters and dynamic routes
- Understanding index routes and layouts

## Lessons

1. [File-Based Routing Concepts](./lesson-01-file-based-routing.md)
2. [Creating Route Components](./lesson-02-route-components.md)
3. [Navigation and Links](./lesson-03-navigation-links.md)
4. [Route Parameters and Dynamic Routes](./lesson-04-route-parameters.md)
5. [Index Routes and Layouts](./lesson-05-index-routes-layouts.md)

## Learning Objectives

By the end of this module, you will:

- Understand file-based routing conventions
- Be able to create route components with type safety
- Navigate between routes using Link components
- Work with dynamic route parameters
- Create nested layouts and index routes
- Organize routes effectively

## Time to Complete

Estimated time: 120-150 minutes

## Prerequisites

Before starting this module, you should have:

- Completed Module 3: Project Structure & Fundamentals
- Basic understanding of React
- A working TanStack Start project

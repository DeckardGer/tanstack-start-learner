# Module 3: Project Structure & Fundamentals

Welcome to Module 3! This module will give you a deep understanding of how a TanStack Start project is organized and how all the pieces fit together.

## Module Overview

In this module, you'll learn:

- The complete directory structure of a TanStack Start project
- What each configuration file does and how to customize it
- How entry points work and the app initialization flow
- The difference between development and production builds
- How to organize your code effectively

## Lessons

1. [Directory Structure Deep Dive](./lesson-01-directory-structure.md)
2. [Configuration Files Explained](./lesson-02-configuration-files.md)
3. [Entry Points and App Structure](./lesson-03-entry-points.md)
4. [Build and Development Modes](./lesson-04-build-modes.md)

## Learning Objectives

By the end of this module, you will:

- Understand the purpose of every directory in a TanStack Start project
- Be able to configure your project for different environments
- Understand the app initialization and rendering flow
- Know when to use development vs production mode
- Be able to organize your project effectively as it grows

## Time to Complete

Estimated time: 90-120 minutes

## Prerequisites

Before starting this module, you should have:

- Completed Module 2: Getting Started
- A working TanStack Start project created
- Basic understanding of React and TypeScript
